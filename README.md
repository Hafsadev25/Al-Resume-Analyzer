# AI Personal Career Coach

An AI-powered resume analyzer built with **Streamlit** and **Google Gemini**.  
Upload your PDF resume and get instant feedback on ATS score, strengths, weaknesses, and keywords.

## Features

- **PDF Upload**: Upload any resume in PDF format
- **AI Feedback**: Get detailed analysis powered by Gemini 1.5 Flash
- **Demo Mode**: Works even without API key - app never crashes
- **ATS Optimization**: Get 10 keywords to beat Applicant Tracking Systems
- **Job Role Specific**: Tailor feedback for any job role
